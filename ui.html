Instructions

The Colour Palettes tab has a tool that allows you to manually pick colours and generate Local Styles for them so they can be easily reused while creating in Figma. After you're happy with your colours, click Create Styles to create local styles.

The Shades tab has a tool that generates shades for you based off of any colour. You can choose one ad-hoc or from your existing styles. After generating some shades, you can turn them into Local Styles styles by clicking Create Styles under the generated shades.

Any colours under Local Styles will be available as code snippets under the Styles Code tab. This tab contains code snippets for web and mobile development (CSS, Android).

If at any time you decide you no longer like the styles you've created, you can manually delete them in Figma. After doing so, click the Refresh button available on one of the tabs to see this reflected. Refresh also works if you manually create styles in Figma and don't see them available.

Example workflow

  1. Go to Colour Palettes and choose some brand colours, or copy and paste existing hex values.
  2. Once colours are picked, click the Create Styles button to see them generated as local styles in Figma. These colours are now available for designing with.
  3. Go to Shades and pick a grey you like. Choose the number of greys you'd like to generate.
  4. Once you're happy with the generated shades, the Create Styles button to see them generated as local styles in Figma. These shades are now available for designing with.
  5. Go to Styles Code and copy any relevant snippets and paste them into your project.
